blinkyboard

A tiny, Arduino-compatible LED driver


View the Project on GitHub

What is this thing?

blinkyboard is a miniscule, open-source device for controlling flexible LED strips. It supports many of the most-common LED strips on the market, including those sold by SparkFun and Adafruit.
